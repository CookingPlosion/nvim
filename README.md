# ⚡ make use of it
- linux 

    **可选步骤**: 先将自己的配置备份 
    
    ~~~sh
    $ cp ~/.config/nvim/ ~/.config/nvim.bak
    ~~~ 
    
    使用以下命令克隆项目 
    
    ~~~sh
    $ git clone https://https://github.com/zhipeng33/nvim.git
    ~~~
